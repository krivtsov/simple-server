# Ruby test task server showing the time

[![Build Status](https://travis-ci.org/krivtsov/simple-server.svg?branch=master)](https://travis-ci.org/krivtsov/simple-server)

## Install

```bash
bundle install
```

or

```bash
rake install
```

## Start sever

```bash
rake start
```

Open <http://localhost:4567>

## Testing application

### Running linter

```bash
rake lint
```

### Running tests

```bash
rake test
```
